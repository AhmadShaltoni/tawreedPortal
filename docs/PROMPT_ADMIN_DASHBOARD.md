# 👨‍💼 PROMPT: Admin Dashboard Developer - Notices Management

## المهمة

تطوير صفحة إدارة الإعلانات في لوحة التحكم تسمح للأدمن بإنشاء وتعديل وتفعيل/تعطيل الإعلانات التي تظهر على الشاشة الرئيسية.

---

## 📍 موقع الصفحة

**Route**: `/admin/notices`

---

## 🎨 مكونات الصفحة

### 1. صفحة القائمة - `page.tsx`

**الموقع**: `app/admin/notices/page.tsx`

**الميزات المطلوبة:**
- عنوان الصفحة: "الإعلانات" / "Announcements"
- زر "إضافة إعلان جديد" (أيقونة + نص)
- جدول يعرض جميع الإعلانات مع الأعمدة:
  - **النص**: أول 50 حرف + "..."
  - **الخلفية**: مربع ملون بالـ hex color
  - **النص اللون**: مربع ملون بالـ hex color
  - **الحالة**: badge أخضر (نشط) / رمادي (معطل)
  - **التاريخ**: تاريخ الإنشاء
  - **الإجراءات**: أزرار (تعديل، تفعيل/تعطيل، حذف)

**الجدول يجب أن يكون:**
- قابل للتصفية (نشط/معطل/الكل)
- قابل للترتيب حسب التاريخ
- متجاوب (responsive) على الموبايل

---

### 2. نموذج الإضافة/التعديل

**الموقع**: `app/admin/notices/new/page.tsx` (إضافة) و `app/admin/notices/[id]/page.tsx` (تعديل)

**الحقول المطلوبة:**

#### أ. نص الإعلان
- **Input**: Textarea (لأنه قد يكون طويل)
- **الحد الأقصى**: 255 حرف
- **عداد الأحرف**: يعرض "120 / 255"
- **الخطأ**: إذا فارغ أو أطول من 255

#### ب. لون الخلفية
- **Input**: Color Picker (native `<input type="color">`)
- **القيمة الافتراضية**: #FFA500 (برتقالي)
- **معاينة حية**: صندوق يعرض اللون المختار

#### ج. لون النص
- **Input**: Color Picker
- **القيمة الافتراضية**: #FFFFFF (أبيض)
- **معاينة حية**: صندوق يعرض اللون المختار

#### د. المعاينة الحية
- **عنوان**: "معاينة" / "Preview"
- **العرض**: يعرض الإعلان كما سيظهر على الشاشة الرئيسية
- **التحديث الفوري**: المعاينة تتغير عند كل تغيير في الحقول

**مثال المعاينة:**
```
┌─────────────────────────────────────┐
│ توصيل مجاني للطلبات أكثر من 50 دينار  │  ← مع الألوان المختارة
└─────────────────────────────────────┘
```

---

### 3. الأزرار والإجراءات

#### أزرار النموذج (Create/Edit):
- **حفظ** (PRIMARY button - أزرق)
- **إلغاء** (SECONDARY button - رمادي) - يرجع للقائمة

#### أزرار الجدول (الإجراءات):
- **تعديل** (EDIT icon - قلم) - ينقل للصفحة Edit
- **تفعيل/تعطيل** (TOGGLE icon - تبديل)
  - إذا النشط → "تعطيل"
  - إذا المعطل → "تفعيل"
- **حذف** (DELETE icon - سلة)
  - يعرض تأكيد قبل الحذف
  - "هل أنت متأكد من حذف هذا الإعلان؟"

## 🎯 التدفق

### إضافة إعلان جديد:
```
1. اضغط "إضافة إعلان جديد"
2. ملء الحقول + معاينة حية
3. اضغط حفظ
4. نجاح → ترجع للقائمة
5. الإعلان يظهر في الجدول
```

### تعديل إعلان:
```
1. اضغط زر التعديل
2. تغيير الحقول + معاينة حية
3. اضغط حفظ
4. نجاح → ترجع للقائمة
```

### تفعيل/تعطيل:
```
1. إعلان نشط → اضغط تعطيل
2. تأكيد لحظي (بدون dialog)
3. الحالة تتغير فوراً في الجدول
```

### حذف:
```
1. اضغط زر الحذف
2. تأكيد في dialog
3. اضغط "حذف"
4. الإعلان يختفي من الجدول (soft delete)
```
